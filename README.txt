This is a Crosshair for no-scoping in CS:GO.
Launch the program using nohair.exe and view the code with nohair.py
Trigger it by using the Up Arrow and disable it with the Down Arrow. 

Requirements for running the Python script:

-Python 3.7 or above
-Pynput (pip install pynput)
-Threading (pip install threading)

Thank you for using nohair!
